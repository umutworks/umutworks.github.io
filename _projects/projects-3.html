The main contribution of this work is an extensive simulations of high accuracy vehicles and weapons systems using Monte Carlo simulations and development of machine learning algorithms for trajectory prediction. In particular:
  1. Developed an algorithm to predict trajectory of adversarial units up to 5 minutes time horizon using Deep Learning and Probabilistic Methods.
  2. Using predicted trajectories implemented a Mixed-Integer Linear Programming algorithm to solve large scale Weapon Task Allocation Problems.
  3. Implemented Monte Carlo simulations of high-fidelity aircrafts, helicopters, missiles and UAVs.
  4. Developed an user interface for adversarial attack simulations and optimization environment for engagement decisions.
